[[req_class_core_modspec]]

[requirement]
====
[%metadata]
identifier:: /req/core/modspec
statement:: A profile of the <<ogc-edr,OGC API - Environmental Data Retrieval Standard>> _SHALL_ be conformant to the <<ogc-modspec,OGC Modular Specification>>.

====

NOTE: The OGC Modular Specification (ModSpec) defines the standard structure for all OGC specifications. It requires that all normative rules be expressed as identifiable Requirements and Recommendations, each of which must be associated with a corresponding Abstract Test Case to ensure the profile is verifiable.
