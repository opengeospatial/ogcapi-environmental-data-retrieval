[[req_nwp-station-level_collectionid]]

[requirement]
====
[%metadata]
identifier:: /req/nwp-station-level/Collectionid
statement:: The Collection id must be based on the center name and the name of the NWP model that the data is sourced from.
part:: the centre name may contain dashes (but no other special characters).
part:: the NWP model name may contain dashes (but no other special characters).
part:: the Collection id SHALL end with the characters `stnlvl`.
part:: The centre name, NWP model name and `stnlvl` SHALL be separated by an underscore `{centre-name}_{NWP model name}_stnlvl`.
part:: The Collection id SHALL be lowercase

[source,text]
----

examples:

ecmwf_ifs_stnlvl
meteofrance_arpege-world_stnlvl
metoffice_global-deterministic-10km_stnlvl
noaa_gfs_stnlvl

----
====
